# Windy Road

[![CI](https://github.com/windyroad/windyroad/actions/workflows/ci.yml/badge.svg)](https://github.com/windyroad/windyroad/actions/workflows/ci.yml)
[![license](https://img.shields.io/github/license/windyroad/windyroad.svg)](https://github.com/windyroad/windyroad/blob/master/LICENSE)

Static website for [windyroad.com.au](https://windyroad.com.au), built with Next.js 15 (static export) and deployed to Netlify.

## Development

```sh
npm run dev
```

Open http://localhost:3000

## Build

```sh
npm run build
```

Static output is generated in the `out/` directory.

## Serve

```sh
npx serve out
```
