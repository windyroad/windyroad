# Leader drafts: The Shift

Drafts of The Shift, the weekly newsletter targeting Engineering Leaders (JOBS_TO_BE_DONE.md J1-J4).

Filename: `YYYY-MM-DD.md`. Generated by `/wr-newsletter persona=leader` (default if no persona arg given). Persona config: `.claude/skills/wr-newsletter/personas/leader.md`.

When published to LinkedIn, create the per-date sub-directory `src/newsletters/published/leader/<publication-date>/` (per ADR-039) and move the brief plus its sibling artefacts (`<publication-date>.md`, `<publication-date>.reviews.md`, `<publication-date>.linkedin.md`, `<publication-date>.capture.md`, `<publication-date>.cover.svg`, `<publication-date>.cover.png`) into that sub-directory.
